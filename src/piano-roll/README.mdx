# Piano Roll

## 機能

- マウスダウンで音を鳴らす
- MIDIキーボードからの入力に対応

### 使用API
- Web Audio API
- Web MIDI API
- Canvas

## 実装概要

- 特に工夫は無しで、地道に頑張るだけ
  - 描画はcanvasで頑張っている
  - 押下判定は座標を頑張っている
  - 音の再生はWeb Audio APIで頑張っている
  - MIDIで音出すのは単にイベントハンドラを登録しているだけ
- しいて言えばCanvasを使っているので、再レンダリングはstate変更でなく命令的な`redraw`関数を実行することで対応していることが工夫（というか単に宣言的でない時代のUIのやり方）をしている
